staticseek is a lightweight and fast full-text search engine designed specifically for static sites. It provides efficient search capabilities without the overhead of traditional search engines, making it ideal for modern web applications.

### Key Features
- **Lightweight**: Minimal footprint, ensuring fast load times.
- **Fast Performance**: Optimized for speed, leveraging advanced indexing techniques.
- **Language Support**: Handles multiple languages, including complex scripts like Japanese.
