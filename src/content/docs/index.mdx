staticseek is a lightweight and fast full-text search engine designed specifically for static sites. It provides efficient search capabilities for any language in the Unicode standard, from English to Arabic, Japanese to Hindi, without the overhead of traditional search engines.

### Key Features
- **Universal Language Support**: Full Unicode compliance enables search in any writing system worldwide.
- **Lightweight**: Minimal footprint, ensuring fast load times.
- **Fast Performance**: Optimized for speed, leveraging advanced indexing techniques.
